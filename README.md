# glow_pong
Pong in pygame

<p>Unzip and and run .exe file to play</p>
<h3>How to play</h3>
<li>Use space to start game</li>
<li>Use arrow keys to change speed of ball</li>
<li>Use enter to turn against robot on and off </li>
